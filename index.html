O objetivo principal é de divulgar esta pagina, junto com meus projetos (incluindo este próprio) ou com projetos que minha participação com outros desenvolvedores.

Qual o meu objetivo para tudo isso ?

Atualmente estou procurando emprego, seria meu primeiro emprego registrado como desenvolvedor. Hoje muitas vagas exigem um alto nível de conhecimento de diversas linguagens e Framework, por esse motivo, que tive a ideia de criar este "Cartão de visita " para mostrar tanto para recrutadores minha experiencia das linguagens usadas, e seu processo no decorrer de cada projeto.

Meu projeto atual.

No momento estou atuando com uma equipe de desenvolvedores, na criação de um site já em produção para a empresa ABEU (https://www.abeu.org.br/), empresa essa que atua no desenvolvimento da cultura editorial universitária, fornecendo soluções produtos e serviços adequados às necessidades dos associados das instituições parceiras e dos leitores, foi desenvolvido o site usando linguagem de programação Python, um micro-framework da linguagem Django chamado Wagtail, a ideia de criar a pagina usando Wagtail é que com ela é dada a liberdade para que o próprio usuário possa ter total acesso na criação de novos usuário e criação de novas paginas e sua edição, sem depender da interferência do desenvolvedor.
