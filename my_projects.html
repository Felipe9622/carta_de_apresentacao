Nome do projeto:ABEU

Sobre o Projeto:

Atuando em colaboração com uma equipe de desenvolvedores, na criação de um site já em produção para a empresa ABEU, empresa essa que atua no desenvolvimento da cultura editorial universitária, fornecendo soluções produtos e serviços adequados às necessidades dos associados das instituições parceiras e dos leitores, foi desenvolvido o site usando linguagem de programação Python, um micro-framework da linguagem Django chamado Wagtail, a ideia de criar a pagina usando Wagtail é que com ela é dada a liberdade para que o próprio usuário possa ter total acesso na criação de novos usuário e criação de novas paginas e sua edição, sem depender da interferência do desenvolvedor.

Linguagens/Ferramentas Utilizadas
  • Python
  • Django
  • Wagtail
  • HTML 5
  • JavaScript
  • CSS
  • Gitlab
  • Docker
Minha experiencia tirada com este projeto
Este projeto foi meu maior desafio de varias maneiras, tanto como meu primeiro projeto oficial para uma empresa, quanto com uma equipe de programadores experientes. Foi a partir dele que tive meu primeiro contato pratico com Docker quanto tambem com Django,Wagtail. Por muitas vezes, para conseguir acompanhar os processos e a equipe, precisei dar tudo de mim quanto aos meus estudos individuais.
